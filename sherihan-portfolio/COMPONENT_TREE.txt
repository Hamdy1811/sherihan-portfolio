ANIMATION SYSTEM

Framer Motion Variants:
│
├─── letterVariants
│    ├─── hidden: { opacity: 0, y: 20 }
│    └─── visible: { opacity: 1, y: 0, delay: i * 0.03 }
│
├─── paragraphVariants
│    ├─── hidden: { opacity: 0, y: 30 }
│    └─── visible: { opacity: 1, y: 0, delay: custom }
│
└─── skillVariants
     ├─── hidden: { opacity: 0, y: 20 }
     └─── visible: { opacity: 1, y: 0, delay: custom }

Scroll Triggers:
- useInView hook with threshold: 0.1
- margin: -100px
- once: true

RESPONSIVE BEHAVIOR

Mobile (< 768px):
├─── Single column layouts
├─── Hamburger navigation
├─── Stacked buttons
└─── Reduced typography scale

Tablet (768px - 1024px):
├─── Two column portfolio grid
├─── Horizontal button layouts
└─── Medium typography

Desktop (> 1024px):
├─── Three column portfolio grid
├─── Full navigation bar
├─── Large typography
└─── Wide containers (max-w-7xl)
